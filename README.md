# One Man Automation Website

This project is the custom website for One Man Automation, built with React, Vite, and TypeScript.

## Project info

A custom AI automation agency website showcasing operational efficiency with AI voice agents, chat agents, and workflow automation.

## Technology Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To work locally:

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```
